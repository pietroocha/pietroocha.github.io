Największy wspólny dzielnik
Największym wspólnym dzielnikiem dwóch liczb naturalnych a, b nazywamy największą liczbę naturalną k taką, że (k|a) oraz (k|b). Oznaczamy ją NWD(a,b). Oto najważniejsze własności tej funkcji:

1. NWD(a,a) = a
2. NWD(a,b) = NWD(b,a)
3. Jeśli a > b to NWD(a,b) = NWD(a-b,b)
4. Jeśli a = k b + r to NWD(a,b) = NWD(b,r)

Algorytm wyznaczania największego wspólnego dzielnika dwóch liczb naturalnych party na tych własnościach został opisany przez Euklidesa w dziale Elementy, w księgach siódmej oraz dziesiątej, które powstało około 300 lat p.n.e.
Pseudo-kod funkcji
a0 = a;
b0 = b;
p = 1; q = 0;
r = 0; s = 1;
while (b != 0)
  c = a mod b;
  quot = floor( a/b )
  a = b;
  b = c;
  r_tmp = r; s_tmp = s;
  r = p - quot * r;
  s = q - quot * s;
  p = r_tmp; q = s_tmp;
